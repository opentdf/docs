# Creating TDFs

A **TDF (Trusted Data Format)** wraps a payload in a cryptographic envelope that binds access control policy directly to the data. The TDF contains an encrypted payload, a manifest describing the policy and key access configuration, and optional assertions.

This page covers how to encrypt and decrypt TDFs, and the full set of options available for both operations.

:::tip Prerequisites
Complete the [Quickstart](/sdks/quickstart) for your language before following these examples. All examples assume a running OpenTDF platform and a configured SDK client.
:::

## Inspecting a TDF Without Decrypting

You can read the manifest and policy of a TDF without performing a key access request or decrypting the payload. This is useful for routing, auditing, or UI display.

<Tabs>
<TabItem value="go" label="Go">

```go
import (
    "bytes"
    "fmt"
    "os"

    "github.com/opentdf/platform/sdk"
)

data, _ := os.ReadFile("sensitive.tdf")

// isTDF check (package-level function)
reader := bytes.NewReader(data)
valid, err := sdk.IsValidTdf(reader)
if err != nil || !valid {
    fmt.Println("Not a valid TDF")
    return
}

// Load and inspect without decrypting payload
reader.Seek(0, 0)
tdfReader, err := client.LoadTDF(reader)
if err != nil {
    log.Fatal(err)
}

manifest := tdfReader.Manifest()
fmt.Printf("TDF version: %s\n", manifest.TDFVersion)

attrs, _ := tdfReader.DataAttributes()
fmt.Printf("Data attributes: %v\n", attrs)

policy, _ := tdfReader.Policy()
fmt.Printf("Policy UUID: %s\n", policy.UUID)
```

</TabItem>
<TabItem value="java" label="Java">

```java
import java.nio.channels.FileChannel;
import java.nio.file.Path;
import java.nio.file.StandardOpenOption;

Path tdfPath = Path.of("sensitive.tdf");

// Check if file is a valid TDF without decrypting
try (var channel = FileChannel.open(tdfPath, StandardOpenOption.READ)) {
    boolean isTdf = SDK.isTDF(channel);
    if (!isTdf) {
        System.out.println("Not a valid TDF");
        return;
    }
}

// Read manifest without decrypting
try (var channel = FileChannel.open(tdfPath, StandardOpenOption.READ)) {
    Manifest manifest = SDK.readManifest(channel);
    System.out.println("TDF version: " + manifest.tdfVersion);

    PolicyObject policy = SDK.decodePolicyObject(manifest);
    System.out.println("Policy UUID: " + policy.uuid);
    for (var attr : policy.body.dataAttributes) {
        System.out.println("Attribute: " + attr.attribute);
    }
}
```

</TabItem>

## Bulk Decrypt (Go)

When decrypting many TDFs in a single operation, use `BulkDecrypt` to batch the KAS rewrap requests and reduce round-trip overhead.

```go
import (
    "bytes"
    "github.com/opentdf/platform/sdk"
)

// Prepare sources
tdfs := []*sdk.BulkTDF{
    {Reader: bytes.NewReader(tdf1Bytes), Writer: &buf1},
    {Reader: bytes.NewReader(tdf2Bytes), Writer: &buf2},
    {Reader: bytes.NewReader(tdf3Bytes), Writer: &buf3},
}

err := client.BulkDecrypt(
    ctx,
    sdk.WithTDFs(tdfs...),
    sdk.WithBulkKasAllowlist([]string{"https://kas.example.com"}),
)
if err != nil {
    // Individual errors per TDF can be extracted:
    errs, ok := sdk.FromBulkErrors(err)
    if ok {
        for i, e := range errs {
            if e != nil {
                fmt.Printf("TDF %d failed: %v\n", i, e)
            }
        }
    }
}
```

For cases where you need to inspect the prepared request before executing, use `PrepareBulkDecrypt`:

```go
prepared, err := client.PrepareBulkDecrypt(ctx, sdk.WithTDFs(tdfs...))
if err != nil {
    log.Fatal(err)
}
// Inspect or modify `prepared` before executing
err = prepared.BulkDecrypt(ctx)
```
