**Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `state` | `ActiveStateEnum` | No | Filter by state: `ACTIVE` (default), `INACTIVE`, or `ANY`. |
| `pagination` | `PageRequest` | No | Pagination options (limit, offset). |

**Example**

<Tabs>
<TabItem value="go" label="Go">

```go
import "github.com/opentdf/platform/protocol/go/policy/namespaces"
resp, err := client.Namespaces.ListNamespaces(context.Background(), &namespaces.ListNamespacesRequest{})
if err != nil {
	log.Fatal(err)
}
for _, namespace := range resp.GetNamespaces() {
	log.Printf("Namespace: %s\n", namespace.GetName())
}
```

**Returns**

A list of `Namespace` objects, each with `id`, `name`, `fqn`, `active` state, and timestamps. Includes pagination metadata.

:::tip Listing inactive namespaces
By default, `ListNamespaces` returns only active namespaces. To include inactive ones, set `state` to `INACTIVE` or `ANY` on the request.
:::
