<summary>Create a Namespace</summary>

**Signature**

<Tabs>
<TabItem value="go" label="Go">

```go
client.Namespaces.CreateNamespace(ctx, &namespaces.CreateNamespaceRequest{...})
```

</TabItem>
<TabItem value="java" label="Java">

```java
sdk.getServices().namespaces().createNamespaceBlocking(req, metadata).execute()
```

</TabItem>
<TabItem value="js" label="JavaScript">

```typescript
await platform.v1.namespace.createNamespace({ ... })
```

</TabItem>
</Tabs>

**Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | `string` | Yes | A valid hostname (e.g., `opentdf.io`, `example.com`). |
| `metadata` | `MetadataMutable` | No | Labels and annotations to attach to the namespace. |

**Example**

<Tabs>
<TabItem value="go" label="Go">

```go
// Imports: namespaces
namespace := &namespaces.CreateNamespaceRequest{
	Name: "opentdf.io",
}

_, err = client.Namespaces.CreateNamespace(context.Background(), namespace)
if err != nil {
	log.Fatal(err)
}
```
