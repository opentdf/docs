otdfctl policy kas-registry get

get - Get a registered Key Access Server

Synopsis

otdfctl policy kas-registry get [flags]

Description

For more information about registration of Key Access Servers, see the manual for kas-registry.

Example​

otdfctl policy kas-registry get --id=62857b55-560c-4b67-96e3-33e4670ecb3b

Options

-i, --id <id>
ID of the Key Access Server registration (required: true)

Aliases

g
Previous
otdfctl policy kas-registry delete
Next
otdfctl policy kas-registry key
